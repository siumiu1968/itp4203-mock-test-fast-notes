下面係最短、最貼 mock test 嘅可 copy 片段。每段都係照你份題目去寫。

Room：Camera Entity

用途：定義一筆 camera 資料長相，對應 `camera_table`。

@Entity(tableName = "camera_table")
data class Camera(
    @PrimaryKey
    val id: String = UUID.randomUUID().toString(),
    val name: String,
    val descriptions: String,
    val imagePath: String
)

Room：CameraDao

用途：首頁查全部、詳細頁查單一資料、Insert page 做 insert。

@Dao
interface CameraDao {
    @Query("SELECT * FROM camera_table")
    fun getAllCameras(): List<Camera>

    @Query("SELECT * FROM camera_table WHERE id = :id LIMIT 1")
    fun getCameraById(id: String): Camera?

    @Insert
    fun insertCamera(camera: Camera)
}

Room：AppDatabase

用途：題目指定 database 名係 `camera_database`，呢段就係個主盒。

@Database(entities = [Camera::class], version = 1, exportSchema = false)
abstract class AppDatabase : RoomDatabase() {
    abstract fun cameraDao(): CameraDao

    companion object {
        @Volatile
        private var INSTANCE: AppDatabase? = null

        fun getDatabase(context: Context): AppDatabase {
            return INSTANCE ?: synchronized(this) {
                val instance = Room.databaseBuilder(
                    context.applicationContext,
                    AppDatabase::class.java,
                    "camera_database"
                )
                    .allowMainThreadQueries()
                    .build()

                INSTANCE = instance
                instance
            }
        }
    }
}

MainActivity：首頁 list + item click

用途：首頁顯示 list，按 `+` 去 Insert，按 item 去 Details。

db = AppDatabase.getDatabase(this)
recyclerView = findViewById(R.id.recyclerView)
fabInsert = findViewById(R.id.fabInsert)

adapter = CameraAdapter(emptyList()) { camera ->
    val intent = Intent(this, DetailsActivity::class.java)
    intent.putExtra("camera_id", camera.id)
    startActivity(intent)
}

recyclerView.layoutManager = LinearLayoutManager(this)
recyclerView.adapter = adapter

override fun onResume() {
    super.onResume()
    val cameraList = db.cameraDao().getAllCameras()
    adapter.updateData(cameraList)
}

InsertActivity：Select Image + Save

用途：最值分位。呢段處理選圖、預覽、save 入 Room。

private var selectedImageUri: Uri? = null

private val imagePicker =
    registerForActivityResult(ActivityResultContracts.GetContent()) { uri ->
        if (uri != null) {
            selectedImageUri = uri
            imageView.setImageURI(uri)
            imageView.visibility = ImageView.VISIBLE
        }
    }

btnSelectImage.setOnClickListener {
    imagePicker.launch("image/*")
}

btnSave.setOnClickListener {
    val name = etName.text.toString().trim()
    val description = etDescription.text.toString().trim()

    if (name.isEmpty()) return@setOnClickListener

    val camera = Camera(
        name = name,
        descriptions = description,
        imagePath = selectedImageUri?.toString() ?: ""
    )

    db.cameraDao().insertCamera(camera)
    finish()
}

DetailsActivity：收 id + 顯示資料

用途：詳細頁讀返同一筆資料，再顯示圖、名、描述。

val cameraId = intent.getStringExtra("camera_id")
if (cameraId != null) {
    val camera = db.cameraDao().getCameraById(cameraId) ?: return
    tvTitle.text = camera.name
    tvDescription.text = "Description: ${camera.descriptions}"

    if (camera.imagePath.isNotEmpty()) {
        imageView.setImageURI(Uri.parse(camera.imagePath))
    }
}

btnBack.setOnClickListener {
    finish()
}
